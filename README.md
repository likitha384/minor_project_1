GroupDNA – Your WhatsApp Group Chat, Decoded-
GroupDNA is a Python-based data analysis project that transforms exported WhatsApp group chats into meaningful insights. Instead of scrolling through thousands of messages, the project automatically analyzes conversations to reveal communication patterns, participant activity, and overall group behavior.
The project reads a WhatsApp chat export, cleans and structures the data, and then performs multiple analyses to generate useful statistics. It identifies the most active members, busiest days and hours, common words and phrases, response times between participants, longest inactive periods, and even includes a fun "Group Mom" score to recognize the most caring or supportive member in the chat.
Built using Python, NumPy, and the datetime module, GroupDNA demonstrates practical applications of file handling, string processing, data structures, date-time manipulation, and basic data analysis. The project is designed to showcase how raw text data can be transformed into valuable insights using simple yet effective programming techniques.
Features
1. Total messages and participant-wise message count
2. Most active group members
3. Busiest days and hours of conversation
4. Most frequently used words
5. Response time analysis
6. Silent streak detection
7. "Group Mom" score for supportive members
8. 24-hour activity heatmap
9. Clear and easy-to-understand chat statistics
This project is ideal for anyone interested in Python, data analysis, text processing, and real-world applications of programming, while also providing a fun way to explore the dynamics of WhatsApp group conversations.
